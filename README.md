Bank Analytics Project

 Overview

This project focuses on analyzing banking data to generate actionable insights that support decision-making and improve overall banking performance.

It highlights key metrics such as loan distribution, transaction methods, branch performance, and customer behavior using data visualization and analytics tools.

 Objectives
 
Analyze banking transactions and customer activity
Identify high-performing and low-performing branches
Evaluate loan distribution and recovery performance
Understand transaction method trends
Build dashboards for business insights

 Key Analysis Areas
 
Customer segmentation (Top & Bottom customers)
Branch-wise transaction analysis
Transaction method distribution (Credit, Debit, Bank Transfer)
Loan performance and outstanding balances
Interest rates and recovery performance

 Tools & Technologies
 
Microsoft Excel
Power BI
MySQL
Python (for data cleaning)

 Key KPIs
 
Total Loan Amount
Total Payment Received
Outstanding Balance
Average Interest Rate

 Key Insights
 
High dependency on debit transactions compared to credit usage
Some branches outperform others in transaction volume
Bank transfer transactions may lead to operational losses
Higher interest rates are linked to higher default rates
Certain regions show lower loan recovery performance

 Challenges Faced

Dynamic KPI creation in Power BI
Data cleaning and handling null values
Data type mismatches (numbers vs text)
Data loading issues in MySQL
Managing relationships and dashboard filters

  Recommendations

Promote credit cards and loan products to debit-heavy users
Optimize bank transfer costs and promote digital payments (UPI)
Improve loan recovery strategies in low-performing branches
Introduce targeted loan schemes for underrepresented age groups
Reduce operational costs and improve backend efficiency
✅ Conclusion

This project demonstrates how data analytics tools can be used to identify financial trends, optimize banking operations, and support strategic decision-making through interactive dashboards and insights.
